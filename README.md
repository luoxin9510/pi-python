# pi-python

Python port of [pi](https://github.com/earendil-works/pi) (earendil-works/pi) — a minimal
coding agent harness, originally written in TypeScript by Mario Zechner.

Phase 1 scope: an embeddable agent SDK (loop + built-in tools + JSONL tree-shaped session
storage). See `docs/superpowers/specs/` for the authoritative design documents.
